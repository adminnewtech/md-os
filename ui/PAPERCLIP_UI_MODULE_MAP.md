# Paperclip UI Module Map v0.1

## Layout
- RTL Arabic-first, English optional
- Sidebar modules + command palette + agent activity drawer
- Company switcher + project switcher in top bar

## Modules
1. Dashboard: KPIs, revenue, open risks, agent status, pending approvals
2. Company Workspace: profile, branches, teams, permissions
3. Project Workspace: roadmap, tasks, files, decisions, agents
4. Agent Studio: create/edit agents, prompts, tools, skills, limits
5. Agent Teams: org chart, responsibilities, run cycles, reports
6. Workflow Studio: visual graph builder, triggers, conditions, runs
7. Task Engine: Kanban, queues, SLA, automation rules
8. Memory Center: semantic facts, session history, consolidation, delete requests
9. Skill Library: reusable workflows, tools, prompts, docs
10. API Connector Hub: integrations, auth status, webhook logs
11. CRM: leads, contacts, deals, pipeline, follow-up automation
12. Support: tickets, SLA, macros, customer health
13. Sales: quotes, proposals, deal desk, approvals
14. Marketing: campaigns, content calendar, social, analytics
15. Finance: invoices, expenses, budgets, cashflow
16. Inventory: SKU, stock, reorder, suppliers
17. HR: employees, recruitment, onboarding, policies
18. Logistics: shipments, vehicles, routes, vendors
19. Developer Studio: repos, PRs, deploys, tests, code review
20. Approval Center: pending approvals by risk/category
21. Security Center: roles, audit logs, secrets, incidents
22. Continuous Improvement: backlog, experiments, postmortems
